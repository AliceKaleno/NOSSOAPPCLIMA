# 🌤️ Nosso App Clima

> Aplicativo desenvolvido para consulta de informações climáticas em tempo real utilizando React Native e integração com APIs.

---

## 📖 Sobre o Projeto

O Nosso App Clima foi desenvolvido com o objetivo de fornecer informações meteorológicas de forma rápida, prática e intuitiva.

A aplicação consome dados de uma API externa para exibir condições climáticas atualizadas, permitindo ao usuário acompanhar informações como temperatura, clima atual e outras métricas relevantes.

Além de aprimorar conhecimentos em desenvolvimento mobile, este projeto permitiu a prática de integração entre front-end e back-end, consumo de APIs e manipulação de dados assíncronos.

---
